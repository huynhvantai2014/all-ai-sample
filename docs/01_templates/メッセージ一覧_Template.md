# メッセージ一覧（IPA標準テンプレート）

| メッセージID | メッセージ種別 | メッセージ内容 | 出現条件 | 備考 |
|--------------|---------------|---------------|----------|------|
| MSG001       | エラー        | ユーザーIDが未入力です。 | ユーザーID未入力で登録ボタン押下時 | 必須入力チェック |
| MSG002       | エラー        | パスワードが不正です。   | パスワード形式不正時              | 形式チェック    |
| MSG003       | 情報          | 登録が完了しました。     | 正常に登録完了時                  |                |
| MSG004       | 警告          | この操作は取り消せません。| 削除ボタン押下時                  | 確認ダイアログ  |

> ※ メッセージID: duy nhất, viết hoa, số thứ tự
> ※ メッセージ種別: エラー (Lỗi), 警告 (Cảnh báo), 情報 (Thông tin), 確認 (Xác nhận), v.v.
> ※ メッセージ内容: nội dung hiển thị cho user
> ※ 出現条件: điều kiện hoặc thao tác sinh ra message
> ※ 備考: ghi chú đặc biệt nếu có
