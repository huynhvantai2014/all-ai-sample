# HR管理システム - ユーザーストーリー

U-RESOURCE-01
- タイトル: ログイン
- ユーザー: 全ユーザー
- やりたいこと: 個人アカウントでシステムにログインしたい
- 目的: セキュリティ確保、認証管理
- 受入基準:
  - ユーザーID・パスワードでのログイン認証
  - JWT認証
  - ログイン成功時のリソース一覧画面への遷移

---

U-RESOURCE-02
- タイトル: リソース一覧・統計表示
- ユーザー: 全ユーザー
- やりたいこと: 全リソースの情報を一覧で確認し、統計・グラフで状況を把握したい
- 目的: リソース状況の把握・分析
- 受入基準:
  - リソースID、名称、状態、部門、稼働率などを一覧表示
  - 各リソース（社員）の詳細情報を最大限表示（氏名、社員ID、部門、職位、スキル、資格、語学レベル、JLPT、年齢、性別、経験年数、年生、年女など）
  - 統計・グラフを画面内で表示（総リソース数、部門別人数、スキル分布、年齢構成、男女比、稼働率、リソース状態別分布、経験年数分布、JLPTレベル分布、資格保有率、部門別稼働率、月次推移グラフ等）
  - 検索・フィルタ機能（任意）
  - ログイン後のみアクセス可能
