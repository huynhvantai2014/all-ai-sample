# HR管理システム - ユーザーストーリー

U-RESOURCE-01
- タイトル: 社員一覧・検索・管理
- ユーザー: HR管理者、管理者
- やりたいこと: 全社員の情報を一覧で確認し、検索・フィルタ・詳細表示・編集を行いたい
- 目的: 効率的な人材管理、社員情報の即座な把握、データドリブンな人事業務
- 受入基準:
  - 社員ID、氏名、メール、部門、職位、ステータス、入社日を一覧表示
  - 氏名・メール・社員IDでキーワード検索可能
  - 部門・ステータス（在職/退職/休暇中）でフィルタリング可能
  - 社員詳細画面への遷移、編集画面への遷移
  - CSV出力機能（実装予定）
  - ページネーション対応、検索結果件数表示
  - JWT認証によるアクセス制御

---

U-RESOURCE-02
- タイトル: HR管理システムへのログイン
- ユーザー: 社員、管理者、マネージャー
- やりたいこと: 個人アカウントでシステムにログインしたい
- 目的: セキュリティ確保、役割ベースアクセス制御、セッション管理
- 受入基準:
  - ユーザーID・パスワードでのログイン認証
  - JWT トークンベースの認証システム
  - 役割（admin/manager/general）ごとの権限管理
  - ログイン成功時のダッシュボードリダイレクト
  - ローカルストレージでのトークン・役割情報保存
  - セッションタイムアウト機能
  - テスト環境用ユーザー種別選択機能
  - 認証エラー時の適切なエラーメッセージ表示

---

U-RESOURCE-03
- タイトル: リソース状況ダッシュボード
- ユーザー: 管理者、マネージャー
- やりたいこと: 全社の人材リソース状況をリアルタイムで把握したい
- 目的: 会社全体のリソース配分状況の可視化、迅速な意思決定支援
- 受入基準:
  - 総社員数、稼働中、空き状況、稼働率の統計カード表示
  - 部門別リソース状況（総数・稼働中・空き状況）の表示
  - プロジェクト参加状況の一覧表示
  - リアルタイムデータ更新（API: /api/resource/status）
  - レスポンシブデザイン対応
  - 他画面（社員一覧、社員追加）への導線
  - ログアウト機能
  - 認証チェックによるアクセス制御

---

U-RESOURCE-04
- タイトル: 新規社員追加・登録
- ユーザー: HR管理者、管理者
- やりたいこと: 新しい社員情報をシステムに登録したい
- 目的: 人材データベースの最新化、新入社員の即座なシステム反映
- 受入基準:
  - 社員ID、氏名、メール、電話番号、部門、職位、ステータス、スキル、入社日の入力
  - React Hook Form による入力フォーム管理
  - メール・社員IDの重複チェック
  - 入力値バリデーション（必須項目チェック、形式チェック）
  - 確認ダイアログによる登録確認
  - 登録成功・失敗の通知機能
  - 登録後の社員一覧画面への遷移
  - CSV/Excel一括インポート機能（実装予定）

---

U-RESOURCE-05
- タイトル: 社員詳細表示・編集
- ユーザー: HR管理者、管理者
- やりたいこと: 個別社員の詳細情報を確認し、必要に応じて編集したい
- 目的: 社員情報の正確な把握、プロフィール管理、データメンテナンス
- 受入基準:
  - 社員の基本情報（ID、氏名、連絡先、部門、職位、ステータス）詳細表示
  - スキル情報の表示（配列形式での表示）
  - 編集画面への遷移機能
  - ステータス（在職/退職/休暇中）に応じた表示
  - 社員一覧画面への戻り機能
  - 存在しない社員IDに対するエラーハンドリング
  - 編集フォームでの既存データ自動入力
  - 更新時の確認ダイアログ
  - 更新成功・失敗の通知機能

---

U-RESOURCE-06
- タイトル: ホーム画面・システム概要
- ユーザー: 未認証ユーザー、新規ユーザー
- やりたいこと: システムの概要を理解し、ログインまたは主要機能にアクセスしたい
- 目的: システムの紹介、ユーザーガイダンス、適切な画面への誘導
- 受入基準:
  - システムの主要機能（人材管理、リソースダッシュボード、効率化）の紹介
  - 認証状態に応じた画面表示（未認証時：ウェルカム画面、認証済み：ダッシュボードリダイレクト）
  - ログインボタンからログイン画面への遷移
  - レスポンシブデザイン対応
  - Material UI による統一されたデザイン
