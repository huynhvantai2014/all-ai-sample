

# ユーザーストーリー（POSシステム）

---

## U-HOME-01: ホーム画面ナビゲーション
- **ユーザー**: 店舗スタッフ
- **目的**: システムのエントリーポイントとして、POSレジ画面へスムーズに遷移したい
- **やりたいこと**: システムロゴ・システム名を確認し、POSレジ画面へワンタッチで移動したい
- **受入基準**:
  - システムロゴ・システム名が表示されている
  - 「POSレジ画面へ」ボタンでPOSレジ画面に遷移できる
  - フッターに著作権表示がある
  - UIはモバイル・タブレット最適化、直感的な操作性

## U-POS-01: POSレジ販売・会計
- **ユーザー**: 店舗スタッフ
- **目的**: 店頭での迅速な販売・会計業務
- **やりたいこと**: 商品を検索・追加し、カート管理・会計処理を簡単に行いたい
- **受入基準**:
  - 商品検索（サジェスト、バーコードスキャン）
  - 商品カート追加・数量調整・削除
  - 合計金額・ポイント自動計算（1%還元）
  - 支払い方法選択（現金・カード・ポイント）
  - 会計・キャンセル操作
  - UIはモバイルファースト、直感的な操作性
  - サンプル商品データで動作確認可能
