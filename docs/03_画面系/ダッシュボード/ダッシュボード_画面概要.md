# ダッシュボード_画面概要

## 目的・業務価値
売上・在庫・顧客データ等の集計・可視化により、経営判断・業務改善を迅速化。現場スタッフ・管理者がリアルタイムで状況把握し、アクションにつなげる。
（ユーザーストーリー: 情報不足, 画面イメージ: ![screen-orders-3.png](../../02_user-stories/screen-orders-3.png)）

## 主な機能
- 売上集計グラフ（日次・週次・月次）
- 在庫状況サマリー
- 顧客数・会員数表示
- 主要KPIダッシュボード
- フィルタ・期間選択
- レスポンシブ対応

## 業務フロー
1. ダッシュボード画面を表示
2. 期間・指標を選択
3. グラフ・サマリーを確認
4. 必要に応じて詳細画面へ遷移

## 利用者
- 管理者
- 店舗スタッフ

## 画面遷移
| 遷移元画面 | 遷移方法         | 条件         | 遷移先画面   |
|------------|------------------|--------------|--------------|
| ホーム画面   | メニュー選択      | 権限あり      | ダッシュボード|
| ダッシュボード| グラフクリック    | なし          | 詳細画面     |

## 画面イメージ
```
┌─────────────────────────────┐
│ [日次][週次][月次]                      │
├─────────────┬─────────────┬─────────────┤
│ 売上         │ 在庫         │ 顧客数       │
│ ─────────── │ ─────────── │ ─────────── │
│ ￥xxxx       │ xx           │ xx           │
│ グラフ       │ サマリー     │ サマリー     │
├─────────────────────────────┤
│ 主要KPI: 売上高/在庫回転率/新規顧客数... │
└─────────────────────────────┘
```

## モバイル・タブレット最適化
- グラフ・サマリーは縦並び表示（スマホ）・横並び（タブレット）
- フィルタ・期間選択は画面上部
- タッチ操作しやすいボタン・グラフ
- 主要指標は大きく・見やすく
