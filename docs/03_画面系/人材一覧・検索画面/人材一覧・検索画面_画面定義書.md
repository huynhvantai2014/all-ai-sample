## API定義（参考・詳細）

### 1. getUserList（GET /api/user/list）
- API ID: API_USER_LIST_001
- エンドポイント: /api/user/list
- メソッド: GET
- 業務機能: 社員リスト・検索
- 入力:
  - keyword: string, 任意, 検索キーワード
  - department: string, 任意, 部門
  - status: string, 任意, 状態
- 出力:
  - userList: array, 社員情報一覧
- エラー:
  - E201: データ取得失敗
- 処理フロー: パラメータ検証→リスト取得→結果返却
- 備考: 主要項目のみ返却。AIによる検索条件自動最適化（最大30%創造性）

### 2. exportUserCSV（GET /api/user/export/csv）
- API ID: API_USER_EXPORT_001
- エンドポイント: /api/user/export/csv
- メソッド: GET
- 業務機能: 社員リストCSV出力
- 入力:
  - department: string, 任意, 部門
  - status: string, 任意, 状態
- 出力:
  - fileUrl: string, CSVファイルURL
- エラー:
  - E202: ファイル生成失敗
- 処理フロー: パラメータ検証→CSV生成→URL返却
- 備考: AIによるデータ集計・分析（最大30%創造性）
