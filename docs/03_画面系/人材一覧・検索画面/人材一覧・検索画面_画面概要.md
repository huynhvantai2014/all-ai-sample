# 人材一覧・検索画面_画面概要

## 1. 業務目的
全社員の状態・部門・プロジェクト情報を一覧・検索・フィルタし、最適な人材配分・管理を実現。

## 2. 主な機能
- 社員リスト表示
- 状態・部門・プロジェクトでフィルタ・検索
- 詳細表示
- CSV/Excel出力
- AIによる検索条件自動提案（最大30%創造性）

## 3. 主業務フロー
1. 管理者が一覧画面を開く
2. フィルタ・検索で対象社員抽出
3. 詳細情報確認
4. 必要に応じてCSV/Excel出力

## 4. 利用者
- 管理者

## 5. 他画面との関係
- ダッシュボード画面（遷移元）
- 新規社員追加画面（遷移先）

## 6. 代表的な利用ケース
- プロジェクト配分前の人材検索
- 状態別の人材リスト抽出

## 7. 特記事項
- モバイル・タブレット最適化（UI/UX）：
  - レスポンシブテーブル
  - タッチ操作対応
  - 検索・フィルタボタン大きめ
  - 優先度順表示、情報量多い場合は折りたたみ
  - 多機能ボタン配置、画面遷移最小化
