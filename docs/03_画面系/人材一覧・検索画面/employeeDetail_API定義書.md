# API定義書（IPA標準）

API名: employeeDetail
API ID: API_EMPLOYEE_DETAIL_001
エンドポイント: /api/employee/detail/{employee_id}
HTTPメソッド: GET
関連画面: 人材一覧・検索画面

## 1. 機能概要
指定された社員IDの詳細情報を取得する。

## 2. 入力パラメータ

| パラメータ名 | 必須 | データ型 | 桁数 | 説明 | 備考 |
|--------------|------|----------|------|------|------|
| employee_id | ○ | string | 20 | 社員ID | URLパス |

## 3. 出力パラメータ

| フィールド名 | データ型 | 桁数 | 説明 | 備考 |
|--------------|----------|------|------|------|
| success | boolean | - | 処理成功フラグ | true/false |
| employee_id | string | 20 | 社員ID | - |
| full_name | string | 100 | 氏名 | - |
| email | string | 255 | メールアドレス | - |
| phone | string | 20 | 電話番号 | - |
| department | string | 50 | 部署 | - |
| position | string | 50 | 役職 | - |
| status | string | 20 | 状態 | - |
| skill | string | 500 | スキル | - |
| join_date | string | - | 入社日 | YYYY-MM-DD形式 |
| created_at | string | - | 登録日時 | ISO8601形式 |
| updated_at | string | - | 更新日時 | ISO8601形式 |

## 4. エラーコード・メッセージ

| エラーコード | メッセージ | 発生条件 |
|--------------|------------|----------|
| E301 | 社員情報が存在しません | 指定された社員IDが存在しない |
| E302 | データ取得失敗 | DB接続エラー等 |
| E303 | 権限不足 | アクセス権限なし |
| E999 | システムエラー | 予期しないエラー |

## 5. 処理フロー
1. 社員ID検証
2. 権限チェック
3. 社員情報取得（DB参照）
4. レスポンス返却

## 6. セキュリティ・その他
- 権限に応じた表示項目制御
- 個人情報保護対応

## 7. テストケース
- 正常ケース: 存在する社員ID
- 異常ケース: 存在しない社員ID、権限なし
