# リソース一覧画面_画面概要

## 業務目的
全リソース（人材）の情報を一覧表示し、統計・グラフで状況を把握する。総リソース数、部門別人数、スキル分布、年齢構成、男女比、稼働率、リソース状態別分布、経験年数分布、JLPTレベル分布、資格保有率、部門別稼働率、月次推移グラフ等。検索、フィルタ、CSVダウンロード、履歴、通知、ヘルプ、権限、レスポンシブ対応。

## 主な機能
- リソース一覧表示（氏名、社員ID、部門、職位、スキル、資格、語学レベル、JLPT、年齢、性別、経験年数、状態、履歴、通知、権限等）
- 統計・グラフ表示（総リソース数、部門別人数、スキル分布、年齢構成、男女比、稼働率、リソース状態別分布、経験年数分布、JLPTレベル分布、資格保有率、部門別稼働率、月次推移グラフ等）
- 検索・フィルタ機能
- CSVダウンロード
- 履歴表示
- 通知表示
- ヘルプボタン
- 権限管理
- レスポンシブ対応

## 主業務フロー
1. ログイン後、リソース一覧画面へ遷移
2. 全リソース情報を表示
3. 必要に応じて検索・フィルタ・統計グラフ・CSVダウンロード・履歴・通知・ヘルプ・権限等を利用

## 利用者
- 全ユーザー
- 管理者
- 一般社員
- ゲスト

## 他画面との関係
- ホーム画面
- ログイン画面
- リソース詳細画面

## 代表的な利用ケース
- 部門別リソース状況の把握
- スキル分布の分析
- 統計・グラフによる分析
- 検索・フィルタ・CSVダウンロード
- 履歴・通知・ヘルプ・権限の確認

## 特記事項
- データ量が多い場合のパフォーマンス考慮
- セキュリティ（認証後のみアクセス可能）
- CSVダウンロード
- 履歴・通知・ヘルプ・権限管理
- レスポンシブデザイン
- アクセシビリティ配慮
- エラー表示
