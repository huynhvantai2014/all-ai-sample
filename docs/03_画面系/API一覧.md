# API一覧

---

## 概要
POSシステム全体で使用されるAPIを画面ごと・業務グループごとに集約し、検索・テスト・開発の効率化を図る。各APIのエンドポイント、メソッド、主な機能、利用画面、詳細定義書へのリンクを明記。

## 画面別API一覧

- カート内容取得API

  - メソッド: GET
  - 主な機能: ユーザーのカート内容取得
  - 利用画面: カート画面

## チェックリスト・要確認事項
- API定義書のリンクが正しいこと
- エンドポイント・メソッド・機能が最新であること
- 画面定義書・API定義書と内容が一致していること
- 情報不足APIは「情報不足」と明記
- 業務変更時は必ずAPI一覧を更新
