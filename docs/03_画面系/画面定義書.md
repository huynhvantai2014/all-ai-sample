## 2. 画面詳細仕様

### S001: ホーム画面
**概要**: システムの最初に表示される画面

**表示内容**:
- 未ログイン時: ウェルカムメッセージ、機能紹介、ログインボタン
- ログイン済み: 自動的にダッシュボードにリダイレクト

**機能**:
- ログイン状態の自動判定
- ログインページへの誘導

**レスポンシブ対応**: 対応

---

### S002: ログイン画面
**概要**: ユーザー認証を行う画面

**入力項目**:
- ユーザーID (必須)
- パスワード (必須)
- ユーザー種別 (開発環境のみ)

**機能**:
- JWT認証
- ロール別アクセス制御
- 認証エラー表示

**バリデーション**:
- 必須項目チェック
- 認証失敗時のエラーメッセージ表示

**テストアカウント**:
- 管理者: admin / 123456
- マネージャー: manager01 / 123456
- 一般社員: emp001 / 123456

---

### S003: ダッシュボード
**概要**: リソース状況とシステム概要を表示

**表示内容**:
- サマリーカード
  - 総社員数
  - 稼働中社員数
  - 空き状況
  - 稼働率
- 部門別状況
- プロジェクト別状況

**機能**:
- リアルタイムデータ表示
- 社員管理ページへのナビゲーション
- ログアウト機能

**権限**: 全ユーザー

---

### S004: 社員一覧画面
**概要**: 社員情報の検索・一覧表示

**表示項目**:
- 社員ID
- 氏名
- メールアドレス
- 部門
- 職位
- ステータス
- 入社日

**検索・フィルター機能**:
- キーワード検索 (氏名、メール、社員ID)
- 部門フィルター
- ステータスフィルター

**操作機能**:
- 詳細表示
- 編集
- 新規登録
- CSV出力 (将来実装)
- ログアウト

**ページネーション**: 対応

---

### S005: 社員詳細画面
**概要**: 個別社員の詳細情報を表示

**表示セクション**:
1. **基本情報**
   - 社員ID
   - 氏名
   - メールアドレス
   - 電話番号

2. **職務情報**
   - 部門
   - 職位
   - 入社日
   - ステータス

3. **スキル情報**
   - 保有スキル一覧

4. **システム情報**
   - 登録日時
   - 最終更新日時

**操作機能**:
- 編集ページへの遷移
- 一覧ページへの戻る
- ログアウト

---

### S006: 社員編集画面
**概要**: 既存社員の情報を編集

**編集可能項目**:
- 氏名 (必須)
- メールアドレス (必須、重複チェック)
- 電話番号
- 部門 (必須、プルダウン選択)
- 職位 (必須、プルダウン選択)
- ステータス (必須、ラジオボタン)
- 入社日 (必須、日付選択)
- スキル (複数選択、自由入力可)

**編集不可項目**:
- 社員ID

**バリデーション**:
- 必須項目チェック
- メール形式チェック
- メールアドレス重複チェック

**操作機能**:
- 保存（確認ダイアログ付き）
- キャンセル
- ログアウト

---

### S007: 社員登録画面
**概要**: 新規社員の情報を登録

**入力項目**:
- 社員ID (必須、手動入力 + 自動生成機能)
- 氏名 (必須)
- メールアドレス (必須、重複チェック)
- 電話番号
- 部門 (必須、プルダウン選択)
- 職位 (必須、プルダウン選択)
- ステータス (必須、デフォルト: 在職)
- 入社日 (必須、デフォルト: 今日)
- スキル (複数選択、自由入力可)

**バリデーション**:
- 必須項目チェック
- メール形式チェック
- 社員ID重複チェック
- メールアドレス重複チェック

**操作機能**:
- 登録（確認ダイアログ付き）
- キャンセル
- 社員ID自動生成
- ログアウト

## 3. 共通仕様

### 3.1 認証・認可
- JWT トークンベース認証
- ローカルストレージでのトークン管理
- 未認証時の自動ログインページリダイレクト

### 3.2 エラーハンドリング
- APIエラーの統一的な表示
- バリデーションエラーの項目別表示
- ユーザーフレンドリーなエラーメッセージ

### 3.3 レスポンシブデザイン
- Material UI による統一されたデザイン
- モバイル対応
- ダークモード非対応（将来実装予定）

### 3.4 国際化
- 日本語表示
- 多言語対応は将来実装予定

### 3.5 ブラウザ対応
- Chrome (推奨)
- Firefox
- Safari
- Edge

## 4. 画面遷移図

```
[ホーム画面] → [ログイン画面] → [ダッシュボード]
                                        ↓
                                [社員一覧画面] ⇄ [社員詳細画面] ⇄ [社員編集画面]
                                        ↓
                                [社員登録画面]
```

### 遷移パターン
- **自動遷移**: 認証状態による自動リダイレクト
- **ユーザー操作**: ボタンクリック、リンククリック
- **フォーム送信**: 登録・更新後のリダイレクト

## 5. 技術仕様

### フロントエンド
- **フレームワーク**: Next.js 16 (App Router)
- **言語**: TypeScript
- **UIライブラリ**: Material UI 5.x
- **状態管理**: React Hooks (useState, useEffect)
- **フォームライブラリ**: React Hook Form
- **HTTPクライアント**: fetch API

### バックエンド
- **API**: Next.js API Routes
- **データベース**: MongoDB
- **ODM**: Mongoose
- **認証**: JWT (jsonwebtoken)
- **パスワードハッシュ**: bcryptjs
