# 画面概要

---

**画面名**：チェックアウト画面
**画面ID**：SRC0004
**画面物理名**：checkout

### 目的・業務価値
商品スキャン、数量・割引調整、合計・税計算を迅速かつ正確に行い、POSでの会計業務を効率化する。

### 主な機能
- 商品バーコード・JANコードスキャン
- 数量・割引調整
- 合計金額・消費税計算
- 商品追加・削除

### 業務フロー
1. レジ担当者が商品をスキャン
2. 数量・割引を調整
3. 合計金額・税額を確認
4. 会計処理を実施

### 利用者
- レジ担当者

### 関連・遷移画面
- 仮販売一覧画面
- 決済方法選択画面

### 代表的な利用シナリオ
- POSでの会計処理

### 特記事項
