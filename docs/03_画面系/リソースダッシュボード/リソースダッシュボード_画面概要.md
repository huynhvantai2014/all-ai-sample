# リソースダッシュボード_画面概要

## 1. 業務目的
会社全体の人材リソース状況（空き・稼働・プロジェクト参加）をリアルタイムで把握し、迅速な意思決定・配分を支援。

## 2. 主な機能
- 部門ごとの空き状況・稼働状況グラフ表示
- プロジェクト参加状況ウィジェット
- フィルタ・カスタマイズ
- CSV/Excel出力
- 警告表示（過剰・不足リソース）

## 3. 主業務フロー
1. 管理者がダッシュボードを開く
2. 状態ウィジェット・グラフで状況確認
3. 必要に応じてフィルタ・期間変更
4. CSV/Excelでデータ出力

## 4. 利用者
- 管理者

## 5. 他画面との関係
- 人材一覧・検索画面（詳細遷移）
- 新規社員追加画面（リソース補充）

## 6. 代表的な利用ケース
- 月次会議でのリソース状況報告
- プロジェクト配分前の状況把握

## 7. 特記事項
- モバイル・タブレット最適化（UI/UX）：
  - レスポンシブグラフ
  - タッチ操作対応
  - 主要指標を集約し情報密度高め
  - ワンタッチでフィルタ・期間変更
  - 多機能ボタン配置、画面遷移最小化
  - AI提案によるダッシュボード自動カスタマイズ（最大30%創造性）
