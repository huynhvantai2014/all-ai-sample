# リソースダッシュボード_画面定義書

## 画面ID・論理名・物理名
- 画面ID: SRC0002
- 画面論理名: リソースダッシュボード
- 画面物理名: dashboard

## 業務目的・業務価値
会社全体の人材リソース状況（空き・稼働・プロジェクト参加）をリアルタイムで把握し、迅速な意思決定・配分を支援。

## 主な機能・業務フロー
- 部門ごとの空き状況・稼働状況グラフ表示
- プロジェクト参加状況ウィジェット
- フィルタ・カスタマイズ
- CSV/Excel出力
- 警告表示（過剰・不足リソース）

### 業務フロー
1. ダッシュボード表示
2. 状態ウィジェット・グラフで状況確認

- レスポンシブグラフ
- タッチ操作対応
- 主要指標集約、情報密度高め
- ワンタッチでフィルタ・期間変更

## 他画面との連携・遷移
- 新規社員追加画面（リソース補充）

## API定義（参考・詳細）

### 1. getResourceStatus（GET /api/resource/status）
- API ID: API_RESOURCE_STATUS_001
- エンドポイント: /api/resource/status
- メソッド: GET
- 業務機能: 部門ごとのリソース状況取得
- 入力:
  - department: string, 任意, 部門名
  - period: string, 任意, 期間
- 出力:
  - statusList: array, リソース状況一覧
- エラー:
  - E101: データ取得失敗
- 処理フロー: パラメータ検証→状況取得→結果返却
- 備考: 主要指標のみ返却。AIによるリソース配分最適化提案（最大30%創造性）

### 2. exportResourceCSV（GET /api/resource/export/csv）
- API ID: API_RESOURCE_EXPORT_001
- エンドポイント: /api/resource/export/csv
- メソッド: GET
- 業務機能: リソース状況CSV出力
- 入力:
  - department: string, 任意, 部門名
  - period: string, 任意, 期間
- 出力:
  - fileUrl: string, CSVファイルURL
- エラー:
  - E102: ファイル生成失敗
- 処理フロー: パラメータ検証→CSV生成→URL返却
- 備考: AIによる自動レポート生成（最大30%創造性）
