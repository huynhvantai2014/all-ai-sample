# 画面概要

---

**画面名**：仮販売一覧画面
**画面ID**：SRC0003
**画面物理名**：provisional_order_list

### 目的・業務価値
仮販売（未確定注文）の一覧を表示し、検索・詳細・確定・取消などの管理を行う。

### 主な機能
- 仮販売一覧表示
- 検索・フィルタ・ソート
- 詳細表示・確定・取消
- 新規・未処理注文のハイライト

### 業務フロー
1. レジ担当者が仮販売一覧画面を開く
2. 仮販売を検索・確認
3. 必要に応じて確定・取消

### 利用者
- レジ担当者

### 関連・遷移画面
- 仮販売作成画面
- チェックアウト画面

### 代表的な利用シナリオ
- レジ担当者が仮販売一覧から注文を確定・取消

### 特記事項
- 新規注文のハイライト
- 状態管理

### 情報不足
なし
