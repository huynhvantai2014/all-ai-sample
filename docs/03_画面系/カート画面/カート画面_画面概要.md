# 画面概要

---

**画面名**：カート画面
**画面ID**：SRC0007
**画面物理名**：cart

### 目的・業務価値
商品リスト（カート）を表示・編集し、合計金額計算や会計処理の準備を行う。

### 主な機能
- 商品リスト表示・編集
- 合計金額・消費税計算
- 商品追加・削除・数量調整
- 空カート時メッセージ表示

### 業務フロー
1. レジ担当者がカート画面を開く
2. 商品リストを確認・編集
3. 合計金額・税額を確認

### 利用者
- レジ担当者

### 関連・遷移画面
- チェックアウト画面
- 商品管理画面

### 代表的な利用シナリオ
- POSでのカート管理

### 特記事項
- 空カート時の明示表示

### 情報不足
なし
