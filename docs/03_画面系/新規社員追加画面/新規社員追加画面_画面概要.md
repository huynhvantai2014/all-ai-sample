# 新規社員追加画面_画面概要

## 1. 業務目的
新しい社員情報を迅速・正確にシステムへ登録し、即時に部門・プロジェクトへ配属できるようにする。

## 2. 主な機能
- 氏名、メール、電話番号、部門、役職、状態、スキル、入社日等の入力
- メール・電話番号重複チェック
- 部門・プロジェクトアサイン
- CSV/Excelインポート
- 追加履歴保存
- AIによる入力補助・自動項目提案（最大30%創造性）

## 3. 主業務フロー
1. 管理者が追加画面を開く
2. 必要情報入力
3. 重複チェック
4. 部門・プロジェクトアサイン
5. 登録・履歴保存

## 4. 利用者
- 人事管理者

## 5. 他画面との関係
- 人材一覧・検索画面（遷移元）
- ダッシュボード画面（遷移元）

## 6. 代表的な利用ケース
- 新入社員の登録
- 部門異動時の情報追加

## 7. 特記事項
- モバイル・タブレット最適化（UI/UX）：
  - レスポンシブフォーム
  - タッチ操作対応
  - 入力欄・ボタン大きめ
  - 必須項目を優先表示
  - 多機能ボタン配置、画面遷移最小化
