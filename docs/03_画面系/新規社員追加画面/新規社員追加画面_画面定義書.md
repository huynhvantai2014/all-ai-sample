## API定義（参考・詳細）

### 1. addUser（POST /api/user/add）
- API ID: API_USER_ADD_001
- エンドポイント: /api/user/add
- メソッド: POST
- 業務機能: 新規社員情報登録
- 入力:
  - email: string, 必須, メール
  - phone: string, 任意, 電話番号
  - department: string, 必須, 部門
  - position: string, 任意, 役職
  - status: string, 必須, 状態
  - skill: string, 任意, スキル
  - joinDate: string, 必須, 入社日
- 出力:
  - userId: string, 登録社員ID
- エラー:
  - E301: 入力不備
  - E302: メール・電話重複
- 処理フロー: 入力検証→重複チェック→登録→結果返却
- 備考: AIによる入力補助・自動項目提案（最大30%創造性）

### 2. importUserCSV（POST /api/user/import/csv）
- API ID: API_USER_IMPORT_001
- エンドポイント: /api/user/import/csv
- メソッド: POST
- 業務機能: 社員情報CSVインポート
- 入力:
  - file: file, 必須, CSVファイル
- 出力:
  - importCount: number, インポート件数
- エラー:
  - E303: ファイル不備
- 処理フロー: ファイル検証→インポート→結果返却
- 備考: AIによるデータ自動分類・エラー補助（最大30%創造性）
