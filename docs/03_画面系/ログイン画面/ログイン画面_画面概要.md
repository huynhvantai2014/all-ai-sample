# ログイン画面_画面概要

## 目的・業務価値
店舗スタッフ・管理者がシステムに安全にログインし、個人認証・権限管理を行う。セキュリティ確保・不正利用防止・業務履歴の正確な記録に寄与。

## 主な機能
- ユーザーID・パスワード入力欄
- ログインボタン
- パスワード表示切替
- エラー表示（認証失敗時）
- レスポンシブ対応

## 業務フロー
1. ユーザーがID・パスワードを入力
2. ログインボタン押下
3. 認証成功でホーム画面へ遷移、失敗時はエラー表示

## 利用者
- 店舗スタッフ
- 管理者

## 画面遷移
| 遷移元画面 | 遷移方法         | 条件         | 遷移先画面   |
|------------|------------------|--------------|--------------|
| ログイン画面 | 初期表示          | なし          | ログイン画面 |
| ログイン画面 | ログイン成功      | 認証OK        | ホーム画面   |

## 画面イメージ
```
┌──────────────────────────────────────────────────────────────────────────────┐
│ ユーザーID: [__________]                                                     │
│ パスワード: [__________] [表示]                                              │
│                                                                              │
│                        [ログイン]                                            │
│                                                                              │
│                エラー: ユーザーIDまたはパスワードが違います                  │
└──────────────────────────────────────────────────────────────────────────────┘
```

## モバイル・タブレット最適化
- 入力欄・ボタンは指で押しやすい大きさ
- エラー表示は画面中央・目立つ色
- キーボード自動表示・入力補助
