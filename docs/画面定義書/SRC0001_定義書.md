## イベント定義 (Event Definition)

### login_btn onClick

- Kiểm tra thông tin đăng nhập, gọi API xác thực

### dev_account onClick

- Tự động điền thông tin test vào form

### username onBlur

- Kiểm tra định dạng tên đăng nhập

### password onBlur

- Kiểm tra định dạng mật khẩu

## Mock/Đối chiếu

- Mock HTML/CSS đã đầy đủ form đăng nhập, checkbox, nút, tài khoản test, hiệu ứng, màu sắc, font hiện đại, đồng nhất.
- Không phát hiện thiếu thành phần nghiệp vụ chính.

## Message/Error

- Hiển thị thông báo lỗi khi đăng nhập sai, tài khoản bị khóa, timeout
- Sử dụng message từ docs/templates/メッセージ一覧_Template.md

## Ghi chú

- Nếu bổ sung chức năng mới (ví dụ xác thực 2 lớp), cần cập nhật lại định nghĩa màn hình và API
- Nếu có điểm chưa rõ, ghi chú "Cần xác nhận" hoặc "レイアウト情報不足"
