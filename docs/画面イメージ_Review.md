# 画面イメージ Review

このファイルは、POSシステム各画面のmock HTML/CSSが画面概要・設計要件に合致しているかを検証したレビュー報告です。

---

## 共通指摘事項（全画面）
- 現状、各画面のヘッダー・ボタン・強調色がバラバラ（青・緑・赤・水色など）。
- ブランド・UI/UX統一性の観点から、色彩設計を統一する必要あり。
- 推奨：メインカラー（例：#1976d2）を全画面のヘッダー・ボタン・強調色に統一。サブカラーは状態や警告のみ使用。
